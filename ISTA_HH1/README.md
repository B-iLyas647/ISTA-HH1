# ISTA HH1 — Système de Gestion des Étudiants et Employés

## 📌 Description

ISTA HH1 est une application web de gestion développée dans le cadre d'un projet de développement digital.

L'objectif de cette application est de faciliter la gestion des données d'un centre de formation en permettant l'administration des informations concernant les étudiants et les employés à travers une interface simple et intuitive.

Le système permet notamment de :
- Gérer les informations des étudiants
- Ajouter et consulter les profils des étudiants
- Gérer les informations des employés
- Afficher des statistiques générales via un tableau de bord dynamique
- Interagir avec une base de données MySQL

## 🚀 Fonctionnalités

### 👨‍🎓 Gestion des étudiants
- Ajout d'un étudiant
- Affichage de la liste des étudiants
- Stockage des informations personnelles et académiques
- Calcul de la moyenne générale

### 👨‍💼 Gestion des employés
- Ajout d'un employé
- Gestion des fonctions et salaires
- Consultation des informations des employés

### 📊 Tableau de bord
- Nombre total d'étudiants
- Nombre total d'employés
- Moyenne générale des étudiants
- Salaire moyen des employés

## 🛠️ Technologies utilisées

- HTML
- CSS
- PHP
- MySQL
- PDO
- XAMPP (Apache/MySQL)

## 🗄️ Base de données

La base de données `ista` contient principalement :

- Table `etudiants`
- Table `employes`

Les données sont manipulées avec des requêtes SQL et une connexion sécurisée via PDO.

## ⚙️ Installation

1. Installer XAMPP ou un serveur PHP/MySQL
2. Copier le projet dans le dossier `htdocs`
3. Importer le fichier `database.sql` dans phpMyAdmin
4. Vérifier les paramètres de connexion dans `connexion.php`
5. Lancer le projet depuis le navigateur `localhost/Folder_name`
